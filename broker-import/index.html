Frequently Asked Questions

What brokers does SignalDeck support?

MT4/MT5 via MetaApi (Elite, with live balance sync), Interactive Brokers via Flex Web Service (all plans), cTrader via OAuth (all plans), SnapTrade aggregator covering Robinhood, Webull, Schwab, Fidelity, E*TRADE, and Coinbase (all plans), plus CSV upload in 8 broker formats including Tradovate and NinjaTrader (all plans, free).

Does the stop loss import automatically?

No — across all import methods, stop loss requires manual entry after import for R-Multiple calculation. Broker data returns fills and P&L, not order-level SL/TP. The import saves time on execution data; the stop loss and trade rationale come from your post-trade review.

What is live balance sync and why does it matter?

MT4/MT5 import via MetaApi reads your live account equity in real time. Kelly Criterion sizing and prop firm drawdown headroom always reflect your actual current balance — not a number you typed in last month. After a drawdown, your sizing adjusts automatically. This is exclusive to the MT4/MT5 integration and the Elite plan.

How does IBKR Flex import work?

Set up a Flex Query in IBKR Account Management (Flex Web Service is free), then either paste your Flex token into SignalDeck's guided connect wizard or upload a Flex XML file directly. SignalDeck handles FIFO round-trip reconstruction for scale-in/out and flip-through-zero positions. Opt-in auto-sync keeps your journal current.

Can I import NinjaTrader or Tradovate trades?

Yes — both are supported via CSV import with fill-level FIFO reconstruction and automatic contract multiplier resolution for correct futures P&L. Export your trade history from NinjaTrader or Tradovate and upload it directly in the import screen.
