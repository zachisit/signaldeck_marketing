Software Engineering Intern (Full Stack)

📍 Remote (20 hrs/week) 💼 Mentorship / Unpaid ⚙️ Full Stack Development

The Mission

We are building an institutional-grade forensic audit layer for retail traders. We're in stealth mode, scaling our infrastructure to support a multi-asset trading environment. We are seeking a high-agency engineer to help maintain our core product, push our new Forex layer across the finish line, and lead the technical architecture for our brand-new Crypto layer.

What You Will Do

  • Core Infrastructure: Maintain and optimize our existing FastAPI backend and Vanilla JavaScript frontend.
  • Multi-Asset Scaling: Finalize the logic, testing, and deployment of the Forex execution layer.
  • Crypto Architecture: Spearhead the backend integration and data modeling for our upcoming Crypto tracking layer.
  • DevOps & Database: Write efficient MariaDB queries and manage Docker containerization within our GitLab CI/CD pipeline.
  • Deploy Automatic Frontend Testing: Lead the conversation and decision making around deploying our frontend testing framework.
  • Founder Collaboration: Work directly with the founder for all technical syncs - getting an under-the-hood look at how to scale a proprietary trading application.

Who You Are

  • The Python Operator: You write clean, efficient Python code and have a solid grasp of building robust APIs with FastAPI.
  • AI-Augmented Developer: You know how to use Claude Code for development. We encourage the use of Agentic workflows and LLM tools to speed up your output, provided the final code quality doesn't suffer.
  • Frontend Capable: You are comfortable working close to the DOM with Vanilla JavaScript - no bloated UI frameworks required to get the job done.
  • Systems Oriented: You have practical familiarity with Docker environments, MariaDB, and version control via GitLab.
  • High Agency: You don't need a 100-step manual. You can take a technical requirement, figure out the roadblocks, and push the solution to production.
  • Asset Literate: You understand the basics of trading. Knowledge of the Crypto ecosystem is a significant plus, though not a strict requirement for the role.

What You Get

  • The War Room Experience: A front-row seat to a startup pivot. You will learn how to ship production code rapidly and efficiently without corporate red tape.
  • Technical Mentorship: Direct insight and code reviews from a Systems Architect with 20 years of experience building distributed systems and automated data pipelines.
