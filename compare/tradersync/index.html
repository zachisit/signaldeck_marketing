SignalDeck vs TraderSync: Futures Support

TraderSync connects to Tradovate, NinjaTrader, and Rithmic — all futures-native platforms. If your broker sends clean fill data, TraderSync surfaces it and P&L shows up correctly. But that P&L is broker-derived: TraderSync displays what the import delivers, not what a contract-spec engine computes. Log an ES trade manually without a broker connection and TraderSync has no way to know it's 2 contracts × $12.50/tick — it treats it like a stock position. SignalDeck fetches the contract spec automatically, computes tick-aligned P&L from first principles, and gates stop-alert evaluation to CME Globex hours. Both work for futures traders using supported brokers — SignalDeck additionally works correctly without any broker import.

Frequently Asked Questions

Does SignalDeck or TraderSync have better backtesting?

SignalDeck has a significantly more sophisticated backtesting workflow. It includes Walk-Forward Analysis for overfitting detection, 1,000-path Monte Carlo simulation for drawdown risk modeling, and grid-search parameter optimization — none of which TraderSync offers. TraderSync has a basic rule-based backtester with 70+ indicators but no Walk-Forward Analysis, no Monte Carlo, and no parameter optimization.

Does TraderSync have R-Multiple tracking?

TraderSync includes R-multiple as one metric among many. SignalDeck is built around R-Multiple as its core organizing principle — every metric (expectancy, SQN score, Kelly Criterion, equity curves, Post Mortem reviews) flows from risk-normalized R-value. The difference is between a metric and a framework.

Does TraderSync support futures trading?

TraderSync can import futures trades from Tradovate, NinjaTrader, and Rithmic and displays the P&L and fill data from those imports. It does not have native tick-aligned P&L math, so manual futures entry or trades from unsupported brokers will use equity-style calculation. SignalDeck fetches the contract spec (tick size, tick value, contract size) automatically for ES, NQ, CL, and GC and computes P&L from first principles — with or without a broker import. CME Globex session monitoring and front-month contract search are also native to SignalDeck. See the SignalDeck Futures page for full details.

Does SignalDeck have broker import now?

Yes — SignalDeck now supports MT4 and MT5 trade import via MetaApi, plus real-time account balance sync so Kelly Criterion sizing always uses your actual current equity. The live balance sync is something TraderSync's import doesn't offer. TraderSync still leads on breadth with 900+ broker integrations for non-MT4/MT5 platforms. Both platforms return fills without stop loss — R-Multiple still requires manual entry after import.

Which is cheaper, SignalDeck or TraderSync?

SignalDeck is free during beta with all features unlocked. TraderSync is $30–$80/month. SignalDeck paid plans launch June 2026 at Pro $30/mo, Elite $50/mo — existing beta testers were grandfathered.
