Stop loss
MetaApi's history-deals endpoint returns fills only — not order-level SL/TP. This is a MetaApi API limitation, not a SignalDeck limitation. Enter your intended stop loss after import to enable R-Multiple calculation.
ℹ️
R-Multiple (auto-calculated once SL is entered)
R = (Exit − Entry) ÷ (Entry − Stop Loss). SignalDeck calculates this automatically the moment you add the stop loss.
Every other MT4/MT5 import tool has the same SL limitation — because it's a constraint of how MetaTrader exposes deal history, not how it's read.

SQN Score

Is Your Edge Real?

SQN = √N × Mean R / StdDev R. Tells you whether your edge is statistically significant or noise. Below 1.6 = poor. 2.0–3.0 = good. Above 3.0 = excellent.

Common Questions

What trading journal supports MT4 import?

SignalDeck supports MT4 and MT5 import via MetaApi. Connect any MT4/MT5 broker account and trades import automatically — fills, P&L, symbol, volume, and execution price. Live account balance syncs in real time for accurate Kelly Criterion sizing. Stop loss requires manual entry after import for R-Multiple calculation, as MetaApi fills don't carry order-level SL/TP data.

Does MT4 import capture stop loss automatically?

No — and this applies to all MT4/MT5 import tools. MetaApi's history-deals endpoint returns fills only, not order-level SL/TP. Stop loss must be entered manually for R-Multiple calculation. Every other tool importing MT4/MT5 trades has the same constraint — it's an API limitation of how MetaTrader exposes deal history.

What is live balance sync and why does it matter?

Live balance sync pulls your real-time balance, equity, margin, and leverage from MetaApi's account-information endpoint. Kelly Criterion and fixed-R sizing recommendations are only accurate if they use your current equity — not a stale manual entry. After a drawdown on a funded account, your balance drops and your position size should adjust immediately. No other trading journal that imports MT4/MT5 trades also syncs your live balance.

Which MT4/MT5 brokers does SignalDeck support?

Any broker that runs a MetaTrader 4 or MetaTrader 5 server — which covers the vast majority of forex, CFD, and prop firm brokers worldwide including FTMO, IC Markets, Pepperstone, Exness, ThinkMarkets, MyFundedFX, The5ers, Apex, and thousands of others. Connection is via MetaApi's cloud connector, which handles compatibility with both MT4 and MT5 server versions.
